Welcome to SABAH'S BUGGY EDITOR

About

This buggy editor allows you to create multiple race buggies which can then be entered into any buggy race. The rules and regulations on what makes a valid buggy are available from the homepage, and they are also available on the 'Make buggy' page in case you forget what the rules are while you are making a buggy. There is no need to panic if you accidentally submit a buggy that violates any of the rules as this buggy will not be saved and the webpage will let you know which rule your buggy violated. If the number of features you are required to fill out for a valid race buggy is too much for you to handle, there is an autofill button at the bottom of the make buggy page which will fill any empty fields with default values. After making a buggy if you realise there is something you would like to change, you can simply go to the 'Show buggy' page from the homepage and then click the 'Edit me' link under whichever buggy you would like to edit.
